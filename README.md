# Marketing Analytics Dashboard

Dashboard interativo de análise de campanhas de marketing digital (Jan 2024 — Fev 2026).

## Métricas

- **Despesas**: Google Ads + Meta Ads
- **MQLs**: Marketing Qualified Leads (Pago + Orgânico)
- **SQLs**: Sales Qualified Leads
- **Custo MQL / Custo SQL**: Eficiência de aquisição
- **Taxa de Conversão MQL → SQL**

## Stack

- HTML5 + CSS3 (sem framework)
- Chart.js 4.4
- Google Fonts (Outfit + IBM Plex Mono)

Hospedado via GitHub Pages.
